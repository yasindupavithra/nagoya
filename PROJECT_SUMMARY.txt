## 📋 What You Need to Do Next

### Step 1: Set Up Firebase Project (10 minutes)
```bash
# Go to Firebase Console: console.firebase.google.com
# 1. Create new project (name: "nagoya-auto-auction")
# 2. Enable Firestore Database (Asia Southeast 1)
# 3. Enable Storage (same region)
# 4. Enable Email/Password Authentication
# 5. Copy Firebase credentials
```

### Step 2: Update Credentials (5 minutes)
```bash
# 1. Edit firebaseConfig.js with your Firebase credentials
# 2. Edit .firebaserc with your project ID
```

### Step 3: Deploy to Firebase (5 minutes)
```bash
npm run build
firebase deploy
```

### Step 4: Create Admin User (2 minutes)
- Go to Firebase Console → Authentication → Users
- Add email/password for owner
- Use those credentials to log in

### Step 5: Add Test Data (5 minutes)
- Sign in at `/admin/login`
- Click "Add Vehicle"
- Fill form and upload images
- Verify vehicle appears on home page

## 🔑 Key Credentials to Set Up

You need to replace these placeholders:

### `firebaseConfig.js`
```javascript
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",           // From Firebase Console
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT.appspot.com",
  messagingSenderId: "YOUR_SENDER_ID",
  appId: "YOUR_APP_ID"
};
```

### `.firebaserc`
```json
{
  "projects": {
    "default": "YOUR_PROJECT_ID"
  }
}
```

## 🆘 Common Issues & Solutions

**Issue:** "Permission denied" uploading images
**Fix:** Run `firebase deploy --only storage:rules`

**Issue:** "User does not exist" on login
**Fix:** Create user in Firebase Console → Authentication

**Issue:** Build fails with TypeScript errors
**Fix:** Run `npm install` again, then `npm run build`
